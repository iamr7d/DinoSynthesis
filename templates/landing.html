The Raup Constraint — Allometric Scaling F_c ∝ M−⅓

The core insight: a neural decoder trained on modern archosaurs will always reproduce modern spectral patterns. It does not know the organism is extinct or 5 tonnes. The Raup Filter enforces physical acoustics:

F_c = F_ref · (M_ref / M)1/3
# F_ref = 8000 Hz @ M_ref = 0.5 kg (Tinamou)

The RaupLowPassLayer applies this as a differentiable soft-sigmoid mask inside the decoder — the model learns allometric constraints end-to-end during fine-tuning.

Brownian Bridge Ensemble — Stochastic Ancestral State Estimation

A single deterministic interpolation ignores the stochastic nature of evolution. The Brownian Bridge gives a probability cloud — instead of one dino_synthesis.wav, we generate 20 samples:

z(t) = (1−t)·z_bird + t·z_croc
       + σ_eff · √(t·(1−t)) · ε

σ_eff = σ · √(T_dino / T_max)
# T_dino=166 Mya, T_max=320 Mya
# σ_eff ≈ 0.43 at t=0.5

Deeper evolutionary time → higher σ_eff → wider latent spread. Each sample represents one plausible ancestral vocalization.

Phylogenetic Distance Matching — Latent Metric Regularisation

Standard VAE latent spaces are not metric — L₂ distance in latent space does not correspond to evolutionary time. The new PhyloDistanceLoss corrects this:

ℒ_phylo = Σ_{i<j} ‖dist(z_i,z_j) − k·T_ij/T_max‖²

Combined with Triplet Anchoring — within-clade variance must be smaller than between-clade distance:

ℒ_trip = max(0, d(a,p) − d(a,n) + margin)

Methodology
01
Data Collection
1,182 MP3s sourced from iNaturalist, Internet Archive, and FreeSound across 11 phylogenetic classes in three anchor groups: Aves (Palaeognathae), Crocodylia (Reptilia), and Mammalia Mass (Whippomorpha, Elephantidae, Phocoidea).
02
Preprocessing
Log-Mel spectrogram patches (128 mels × 256 frames) normalized to [0,1]. Each recording chunked into non-overlapping 3-second windows. SNR grading (A/B/C/D) used to quarantine grade-D noise fills.
03
Phylogenetic Parity
Strategic undersampling capped all groups at the Reptile floor (426 tensors). This eliminated the 24:1 Bird-to-Reptile bias present in the raw dataset, improving validation reconstruction loss from 0.0714 → 0.0655.
04
β-VAE Architecture
4-stage residual encoder with Squeeze-Excitation, 128-dim latent space, β = 2.0 KL pressure, free-bits floor 0.5 nats/dim, 40-epoch KL warm-up. Group-weighted sampler ensures equal batch representation.
05
Latent Blending
Per-class centroids computed from 200 encoded μ-vectors each. Blended latent: z = Σ wi·ci / Σ wi. Griffin-Lim (128 iters) with optional Sobel sharpening and latent jitter.
06
Synthesis
The midpoint zdino = 0.5·zbird + 0.5·zcroc represents the hypothetical ancestral archosaur vocal anatomy prior to avian syrinx and crocodilian glottis specialization.
Core Synthesis Logic
# Blend latent centroids by phylogenetic weight
z_blend = (0.5 * centroids["bird"]
         + 0.5 * centroids["croc"])

# Decode from latent midpoint → spectrogram
spec = model.decode(z_blend.unsqueeze(0))

# Griffin-Lim inversion → waveform
wav = hifi_griffinlim(spec, n_iter=128, gate=0.05, sobel_strength=1.0)
